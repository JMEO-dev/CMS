# CMS
Content Management System

# Live Demo
Dashboard: https://cms-ashen-phi.vercel.app/dashboard/products
View and manage product inventory, including details like name, SKU, category, price, stock, status, satisfaction, and sales trend.


# Prerequisites
Node.js (v18 or later)
npm, Yarn, or pnpm

# Getting Started
1. Clone the Repository
git clone git@github.com:JMEO-dev/e-store.git
cd client 

2. Install Dependencies
npm install
# Or: yarn install
# Or: pnpm install

3. Run the Development Server
npm run dev
# Or: yarn dev
# Or: pnpm dev

Visit http://localhost:3000 to see the app

# Scripts
npm run dev: Start development server
npm run build: Build for production
npm run start: Run production build
npm run lint: Run ESLint (if configured)
